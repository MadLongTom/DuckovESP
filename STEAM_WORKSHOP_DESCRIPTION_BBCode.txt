[h2]Main Features / 主要功能[/h2]

[h3]📍 ESP Transparency System (ESP) / 📍 透视系统 (ESP)[/h3]
[list]
[*]Real-time display of chest, dropped item, quest item locations / 实时显示箱子、掉落物品、任务物品位置
[*]Enemy transparency: display enemy location, health, weapon, equipment value / 敌人透视：显示敌人位置、血量、武器、装备价值
[*]3D labels: display distance, name, status in game screen / 3D标签：游戏画面中显示距离、名称、状态
[*]Connection lines: visual connection lines from player to target / 连线功能：从玩家到目标的可视连线
[*]Mini-map markers: automatically mark important items on mini-map / 小地图标记：自动在小地图上标记重要物品
[/list]

[h3]🎯 Auto Aim (Aimbot) / 🎯 自动瞄准 (Aimbot)[/h3]
[list]
[*]Intelligent lock: automatically lock closest enemy head or body / 智能锁定：自动锁定最近的敌人头部或身体
[*]Smooth tracking: adjustable smoothness, simulates real aiming / 平滑跟踪：可调节的平滑度，模拟真实瞄准
[*]Movement prediction: calculate target movement trajectory in advance / 预测移动：提前计算目标移动轨迹
[*]Wall penetration option: choose whether to aim at targets behind walls / 穿墙选项：可选择是否瞄准墙后目标
[*]Bullet redirection: automatically correct bullet trajectory after shooting / 子弹重定向：射击后自动修正子弹轨迹
[*]No Recoil: stable aiming / 无后座力：稳定瞄准
[/list]

[h3]⚡ Auto Trigger (TriggerBot) / ⚡ 自动扳机 (TriggerBot)[/h3]
[list]
[*]Intelligent fire: automatically shoot when crosshair is on enemy / 智能开火：检测到准星对准敌人时自动射击
[*]Bullet teleportation: bullet teleports directly to target vicinity, ignores walls / 子弹传送：子弹直接传送到目标附近，无视墙体
[*]Delay setting: adjustable trigger delay, avoid detection / 延迟设置：可调节触发延迟，避免被检测
[*]Aiming trigger: choose to trigger only when holding right click / 瞄准触发：可选择仅在右键瞄准时触发
[*]Teammate protection: automatically identify teammates, avoid friendly fire / 友军保护：自动识别队友，避免误伤
[/list]

Traditional method: Gun muzzle → [Wall blocks] ❌ → Target
传送方式：枪口 → [瞬移] ✅ → 目标附近 → 目标

[list]
[*]Bullets spawn 1-2 meters in front of target / 子弹在目标前方 1-2 米生成

[*]Automatically set direction and velocity / 自动设置方向和速度
